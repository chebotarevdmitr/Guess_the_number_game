  srand(static_cast<unsigned int>(time(0))); // Инициализация генератора случайных чисел
time(0):

   Функция time из библиотеки <ctime> возвращает текущее время в секундах с начала эпохи (1 января 1970 года). 
Если передать 0 в качестве аргумента, функция вернет текущее время.
     Это значение будет постоянно меняться, что делает его подходящим для инициализации генератора случайных чисел, 
чтобы каждый раз при запуске программы генерировались разные последовательности случайных чисел.

 static_cast<unsigned int>(...):
     static_cast — это оператор приведения типов в C++. Он используется для явного преобразования типа.
    Здесь мы приводим результат функции time(0) к типу unsigned int, чтобы убедиться, что передаем значение подходящего типа в функцию srand.
   
     Функция srand (seed random) используется для инициализации генератора случайных чисел. Она устанавливает начальное значение (или "  ") 
для генератора случайных чисел. Это значение определяет последовательность случайных чисел, которые будут генерироваться функцией rand.
     Если не инициализировать генератор с помощью srand, функция rand будет возвращать одну и ту же последовательность случайных 
чисел при каждом запуске программы.

 rand():
   Функция rand возвращает случайное целое число. По умолчанию оно может быть в диапазоне от 0 до RAND_MAX, где RAND_MAX — это константа, 
определяющая максимальное значение, которое может вернуть функция rand.

 rand() % 100:
    Оператор % (остаток от деления) используется для ограничения диапазона случайных чисел. В данном случае, rand() % 100 
будет возвращать случайное число от 0 до 99.

 + 1:
    Чтобы получить диапазон от 1 до 100, мы добавляем 1 к результату. Таким образом, итоговое значение переменной secretNumber
будет находиться в диапазоне от 1 до 100 (включительно).
 int guess = 0;: 

  Переменная guess инициализируется нулем и будет использоваться для хранения предположения игрока о загаданном числе.
 Переменная attempts также инициализируется нулем и будет отслеживать количество попыток, которые игрок сделал для угадывания числа.
